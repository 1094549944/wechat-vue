# 51jyfw-wechat

> 51教育服务微信项目

## Develope with

> Vue.js

> Vue-router

> Vue-resource

> Vue-touch

> Mint-UI

> Babel

> Webpack

> Npm

> Vuex

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```

##Router Config解读
###name
路由名称，可以通过name跳转路由
###title
设置页面标题
###belong
设置页面导航焦点，匹配导航菜单id：home,action,klass,analysis,exercise
